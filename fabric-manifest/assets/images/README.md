# Assets for Fabric Mapping Service

This directory contains the visual assets required for the Microsoft Fabric Mapping Service workload.

## Required Images

The following image files should be placed in this directory:

### Workload Icons
- `FabricMappingService_Icon.png` - Main workload icon (used in Fabric UI)

### Item Type Icons
- `ReferenceTableItem_Icon.png` - Icon for Reference Table item type
- `MappingConfigurationItem_Icon.png` - Icon for Mapping Configuration item type
- `MappingJobItem_Icon.png` - Icon for Mapping Job item type

### Learning Materials Images
- `MappingService_Hub_GetStarted_Docs.png` - Image for "Getting Started" documentation
- `ReferenceTable_Guide.png` - Image for "Reference Tables Guide" documentation
- `AttributeMapping_Guide.png` - Image for "Attribute Mapping Guide" documentation

## Image Specifications

- **Format**: PNG (Portable Network Graphics)
- **Recommended Size**: 
  - Icons: 48x48 pixels for small icons, 128x128 pixels for large icons
  - Learning Materials: 400x300 pixels or similar aspect ratio
- **Background**: Transparent background recommended for icons
- **Color Scheme**: Should align with Microsoft Fabric design language
